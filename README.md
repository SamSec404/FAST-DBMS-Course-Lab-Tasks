# 📖 About This Repository

This repository contains all laboratory work completed during the **Database Management Systems (DBMS)** course. The labs cover both theoretical concepts and practical implementations, starting from database fundamentals and SQL to advanced topics such as transactions, stored procedures, triggers, and NoSQL databases.

The purpose of this repository is to:

* Strengthen practical database skills.
* Maintain an organized collection of lab work.
* Serve as a learning resource and revision guide.
* Demonstrate hands-on experience with database technologies.
